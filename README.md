# next-app-router-template

## 技術スタック

- 状態管理：Redux
- CSS フレームワーク：tailwind
- データフェッチ：SWR
- テスティングフレームワーク : jest
- コンポーネント管理：storybook

## フォルダ構成

| ルート   |              |                                         |
| -------- | ------------ | --------------------------------------- |
| public   | -            | アセット                                |
| src      | app          | ルーティング等                          |
|          | components   | コンポーネントファイル（atomic design） |
| 　　　　 | constants    | 定数                                    |
| 　　　　 | feature      | 機能                                    |
| 　　　　 | hooks        | カスタムフック                          |
|          | libs         | ライブラリの初期設定等                  |
|          | repositories | データフェッチ                          |
|          | stores       | 状態管理                                |
|          | types        | 型定義                                  |
